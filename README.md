# Study Planner (C++)

## Overview
This is a console-based Study Planner built using C++. It helps students organize their study schedule efficiently by considering subject importance, difficulty, deadlines, and available study time. The system generates a structured daily plan for balanced preparation.

## Features
- Add, edit, and delete subjects
- Mark subjects as completed
- Set total study days and daily hours
- Generate an optimized day-wise schedule
- Display plan in a tabular format
- Automatically allocate time for revision

## Scheduling Logic
- Ensures minimum coverage for each subject
- Distributes remaining time based on priority (importance / difficulty)
- Allocates time evenly across all days
- Uses extra time for revision

## Technologies Used
- C++
- Standard Template Library (STL)
- Console-based interface

## Usage
Run the program, add subjects with required details, set study time, and generate the schedule. You can modify subjects and track progress anytime.

## Menu Options
1. Add Subjects
2. Set Study Time
3. Generate Plan
4. Mark Completed
5. Edit Subject
6. Delete Subject
7. Show Subjects
0. Exit

## Use Case
Useful for students preparing for exams who want a simple and automated way to manage their study schedule.
